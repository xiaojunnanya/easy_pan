## 网盘项目

个人网盘项目

后端部分：[kbws13/Netdisk](https://github.com/kbws13/Netdisk)

个人博客：www.xiaojunnan.cn



## 前提

- node版本：16.16.0
- 安装环境：`npm i`
- 运行：`npm start`
- React + TS
